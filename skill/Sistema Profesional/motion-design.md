\# Motion Design



\## Objetivo



Utilizar movimiento para explicar.



Nunca para decorar.



\---



\# Principio Fundamental



Toda animación debe tener propósito.



Si no comunica:



Eliminar.



\---



\# Duraciones



Microinteracciones



150ms - 250ms



Hover



200ms - 300ms



Transiciones



300ms - 500ms



Scroll Reveal



400ms - 700ms



\---



\# Animaciones Permitidas



Fade



Fade Up



Scale



Slide



Stagger



Parallax Suave



\---



\# Scroll Storytelling



El scroll debe sentirse como una narrativa.



Cada sección aparece cuando es necesaria.



\---



\# Hover



Todo elemento interactivo debe responder.



Ejemplos:



\- elevación

\- brillo suave

\- escala 1.02



\---



\# Parallax



Usar únicamente en:



\- Hero

\- Fondos

\- Elementos decorativos



Velocidad:



5% - 20% más lenta que el scroll.



\---



\# Prohibido



\- rebotes exagerados

\- animaciones infinitas

\- zoom agresivo

\- rotaciones innecesarias

\- movimientos distractores



\---



\# Performance



Animar usando:



transform



opacity



Evitar:



top



left



width



height



\---



\# Auditoría



\- ¿La animación tiene propósito?

\- ¿La experiencia parece premium?

\- ¿La navegación es fluida?
